#Momentum

Robotic arm competition.

##Content

- Code for controling the robotic arm with inverse kinematics
- Code for EMG signal cleaning and encoding

##Arm Specs

###Dimensions:

L1 - 14.5 cm
L2 - 18.75 cm
L3 - 7.3 cm

###Servo pulse limits and angle limits:

Base - 210° -> 130, 600
5 - 190° -> 140, 580
4 - 175° -> 150, 510
3 - 135° -> 300, 630
2 - 180° -> 120, 740
1 - 2.54 cm -> 110, 400
